# Исходники диаграмм

Файлы в этой папке можно открыть в онлайн-редакторах и экспортировать в PNG/SVG:

- `.mmd` — https://mermaid.live
- `.puml` — https://www.plantuml.com/plantuml
- `.dbml` — https://dbdiagram.io

После экспорта изображения рекомендуется положить рядом с исходниками и добавить ссылки на них в README соответствующего этапа.
