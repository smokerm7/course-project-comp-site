# Архитектурные решения ADR

## ADR-001. PCMEF

Принято решение использовать PCMEF, чтобы разделить UI, контроллеры, бизнес-логику, сущности и доступ к данным.

## ADR-002. React + TypeScript

Frontend реализуется на React 18 и TypeScript, так как это соответствует требованиям веб-траектории и упрощает создание SPA.

## ADR-003. Spring Boot

Backend реализуется на Spring Boot 3.x, потому что фреймворк поддерживает REST API, безопасность, JPA и Swagger.

## ADR-004. PostgreSQL

Для хранения данных используется PostgreSQL, так как предметная область имеет связанную реляционную структуру.

## ADR-005. JWT

Авторизация выполняется через JWT, чтобы frontend мог хранить токен и отправлять его в заголовке Authorization.
