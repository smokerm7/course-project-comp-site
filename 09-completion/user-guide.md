# Руководство пользователя

1. Зарегистрироваться или войти в систему.
2. Открыть каталог и выбрать товар.
3. Добавить товар в корзину.
4. Оформить заказ.
5. Для ремонта открыть раздел «Ремонт», заполнить тип устройства, модель и описание проблемы.
6. В личном кабинете отслеживать заказы и заявки.
