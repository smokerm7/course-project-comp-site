# Итоговый отчёт по курсовому проекту

## Тема

Разработка веб-приложения для компьютерной фирмы **MS7-COMP**.

## Автор

**Хизриев Магомед-Салах Алиевич**

Институт: **Институт перспективной инженерии**

## Цель проекта

Целью курсового проекта являлась разработка веб-приложения для автоматизации деятельности компьютерной фирмы, занимающейся продажей компьютерной техники и обработкой заявок на ремонт устройств.

## Выполненные задачи

1. Проведён бизнес-анализ предметной области.
2. Сформирован глоссарий терминов.
3. Описаны акторы и варианты использования.
4. Разработана архитектура приложения на основе PCMEF.
5. Спроектирована база данных.
6. Реализован backend на Spring Boot.
7. Реализован frontend на React + TypeScript.
8. Настроена JWT-аутентификация и ролевая модель.
9. Реализованы каталог товаров, корзина, заказы и заявки на ремонт.
10. Подготовлена документация, руководства пользователя и администратора.

## Реализованный функционал

- регистрация пользователей;
- авторизация по JWT;
- разграничение доступа по ролям;
- каталог компьютерной техники;
- поиск и фильтрация товаров;
- сортировка товаров по цене;
- корзина;
- оформление заказа;
- создание заявки на ремонт;
- административная панель;
- Swagger-документация API.

## Использованные технологии

| Компонент | Технологии |
|---|---|
| Frontend | React 18, TypeScript, Vite, Axios |
| Backend | Java 17, Spring Boot 3.x |
| Безопасность | Spring Security, JWT, BCrypt |
| База данных | H2, PostgreSQL |
| API | REST, OpenAPI/Swagger |
| Архитектура | PCMEF |

## Результат

В результате был разработан рабочий прототип веб-приложения **MS7-COMP**, соответствующий требованиям курсового проекта по веб-разработке.

## Вывод

Курсовой проект позволил закрепить навыки анализа предметной области, проектирования требований, разработки архитектуры, проектирования базы данных и реализации веб-приложения с использованием современных технологий backend и frontend-разработки.
