# Расширенный глоссарий

1. Пользователь — учётная запись в системе.
2. Клиент — пользователь, оформляющий заказы и ремонт.
3. Сотрудник — пользователь, обрабатывающий операции фирмы.
4. Администратор — пользователь с полными правами.
5. Каталог — список товаров.
6. Товар — позиция для продажи.
7. Категория — группа товаров.
8. Корзина — временный набор товаров.
9. Заказ — подтверждённая покупка.
10. Позиция заказа — товар в заказе.
11. Цена — стоимость товара.
12. Остаток — количество товара на складе.
13. Ремонт — услуга восстановления техники.
14. Заявка — обращение клиента.
15. Устройство — техника клиента.
16. Диагностика — определение неисправности.
17. Статус — состояние процесса.
18. Комментарий мастера — примечание сотрудника.
19. Оплата — фиксация платежа.
20. Доставка — передача товара клиенту.
21. Роль — уровень доступа.
22. JWT — токен авторизации.
23. BCrypt — хеширование пароля.
24. REST API — интерфейс клиента и сервера.
25. DTO — объект передачи данных.
26. Entity — JPA-сущность.
27. Repository — доступ к данным.
28. Service — бизнес-логика.
29. Controller — обработчик HTTP-запросов.
30. Swagger — документация API.
31. PostgreSQL — реляционная БД.
32. React — frontend-библиотека.
33. Spring Boot — backend-фреймворк.
