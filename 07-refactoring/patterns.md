# Паттерны

- Repository — доступ к данным через Spring Data JPA.
- DTO — отделение API-моделей от Entity.
- Service Layer — бизнес-логика вне контроллеров.
- Lazy Load — отложенная загрузка связей JPA.
- Data Mapper — преобразование Entity в DTO.
