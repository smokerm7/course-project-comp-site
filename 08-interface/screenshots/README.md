# Скриншоты интерфейса

В эту папку необходимо добавить скриншоты работающего веб-приложения перед загрузкой на GitHub или сдачей курсового проекта.

Рекомендуемые файлы:

| Файл | Экран |
|---|---|
| `home.png` | Главная страница |
| `catalog.png` | Каталог товаров |
| `product.png` | Карточка товара |
| `cart.png` | Корзина |
| `repair.png` | Форма заявки на ремонт |
| `profile.png` | Личный кабинет клиента |
| `admin.png` | Административная панель |
| `swagger.png` | Swagger UI |

Скриншоты можно сделать через браузер после запуска проекта:

- Frontend: `http://localhost:5173`
- Backend Swagger: `http://localhost:8081/swagger-ui/index.html`
