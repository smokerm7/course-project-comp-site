# Безопасность

- Пароли хранятся через BCrypt.
- После входа пользователь получает JWT.
- Токен передается в заголовке `Authorization: Bearer <token>`.
- Роли: CLIENT, EMPLOYEE, ADMIN.
- CORS настроен для frontend `http://localhost:5173`.
