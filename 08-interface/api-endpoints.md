# REST API

| Метод | Endpoint | Описание |
|---|---|---|
| POST | /api/auth/register | Регистрация |
| POST | /api/auth/login | Вход |
| GET | /api/products | Каталог товаров |
| GET | /api/products/{id} | Карточка товара |
| GET | /api/categories | Категории |
| POST | /api/orders | Создание заказа |
| GET | /api/orders/my | Заказы клиента |
| POST | /api/repairs | Создание заявки |
| GET | /api/repairs/my | Заявки клиента |
| PATCH | /api/repairs/{id}/status | Изменение статуса ремонта |
| POST | /api/admin/products | Добавление товара |
| PUT | /api/admin/products/{id} | Редактирование товара |
| DELETE | /api/admin/products/{id} | Удаление товара |
