# Бизнес-глоссарий

| Термин | Определение |
|---|---|
| Клиент | Пользователь, который покупает товары или оформляет ремонт |
| Сотрудник | Работник фирмы, обрабатывающий заказы и заявки |
| Администратор | Пользователь с полными правами управления системой |
| Товар | Компьютерная техника или комплектующее для продажи |
| Категория | Группа товаров: ноутбуки, ПК, видеокарты, мониторы |
| Каталог | Раздел веб-приложения со списком товаров |
| Корзина | Временный список выбранных клиентом товаров |
| Заказ | Оформленная покупка одного или нескольких товаров |
| Позиция заказа | Конкретный товар внутри заказа с количеством и ценой |
| Остаток | Количество товара на складе |
| Ремонтная заявка | Обращение клиента по поводу неисправного устройства |
| Устройство | Техника клиента: ноутбук, ПК, монитор или комплектующее |
| Диагностика | Проверка устройства для определения неисправности |
| Статус ремонта | Текущее состояние ремонтной заявки |
| Статус заказа | Текущее состояние заказа клиента |
| Личный кабинет | Раздел клиента с заказами и заявками |
| REST API | Интерфейс взаимодействия frontend и backend |
| JWT | Токен авторизации пользователя |
| BCrypt | Алгоритм хеширования паролей |
| Swagger | Документация REST API |
