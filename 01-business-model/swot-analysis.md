# SWOT-анализ

| Сильные стороны | Слабые стороны |
|---|---|
| Единая база клиентов, товаров и заявок | Требуется поддерживать актуальность каталога |
| Прозрачный контроль ремонта | Нужно обучить сотрудников работе с системой |
| Онлайн-оформление заказов | Зависимость от сервера и сети |
| Разграничение доступа по ролям | Первичная разработка требует времени |

| Возможности | Угрозы |
|---|---|
| Добавление онлайн-оплаты | Конкуренция с крупными магазинами |
| Уведомления клиентам | Ошибки пользователей при вводе данных |
| CRM-функции и отчёты | Утечка данных при слабой защите |
| Расширение до интернет-магазина | Технические сбои |
