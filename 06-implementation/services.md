# Сервисный слой

Сервисный слой содержит бизнес-логику: проверку остатков, создание заказов, назначение статусов ремонта, хеширование паролей и работу с JWT.

| Сервис | Назначение |
|---|---|
| AuthService | Регистрация, вход, генерация токена |
| ProductService | Работа с товарами и категориями |
| OrderService | Оформление заказов |
| RepairService | Создание и обработка заявок |
| ReportService | Отчёты по продажам и ремонтам |
